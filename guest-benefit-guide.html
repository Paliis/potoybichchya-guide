Шлях / вигода гостя в Потойбіччі

Що гість отримує від гри й участі — і що його утримує. По сегментах: Автопілот, Casual, Активний, Хардкор. Окремо — додаткові бенефіти з балансу гри.

Ризик Автопілоту: після онбордингу й першого використання СП — відвал; наступних ігрових СП немає. Луп у новий сезон не замикається.

Очікувана вигода гостя окремий шар

Головне — грн/міс. Сезон коротко поряд. Старт СП тут з сегментним % погашення (не середнє 2,85 на всіх F4). Ігрові СП — з балансу/лупів; призи — алокація T3. Вартість для Silpo — економіка гостя.

Розклад для обраного сегмента

Де розходяться сегменти

Після спільного онбордингу (до використання СП). Шкала інтересу якісна (немає / мін. / середнє / високе) — без % гостей і без вартості для Silpo.

Ще з балансу гри окремий шар

Механіки, яких немає окремим кроком на піді вище. Релевантність для обраного сегмента — якісно.
